
# heyao-query - 河妖工房订单查询插件

## 功能特点

- 查询订单变化信息
- 实时返回处理后的订单图片

## 使用方法

只需安装插件，当用户发送/heyao <订单号>时，插件会自动响应并发送处理后的图片。

## 安装方法

1. 在插件市场中填入仓库链接:https://github.com/AmeiMiao/astrbot_plugin_heyao-query

## 依赖

- Pillow
